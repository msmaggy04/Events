# Events

A lightweight web app prototype to search for **local, in-person events** by:

- ZIP code
- date range
- search radius (miles)
- event type(s)

Online-only events are automatically excluded from search results.

## Run locally

Open `index.html` in any modern browser.

## Notes

- This project uses a small in-file demo dataset in `app.js`.
- Supported demo ZIP codes: `10001`, `30301`, `60601`, `73301`, `94103`.
